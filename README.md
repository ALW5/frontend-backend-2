Fullstack-приложение интернет-магазина мебели с REST API и Swagger документацией.

## Отчеты по практическим работам

* **Практическая работа №1**
  Реализовал карточку товара (содержимое: название, описание, фотография) с применением переменных (минимум 2), миксина (минимум 1) и использованием вложенной структуры селекторов на препроцессоре SASS.

* **Практическая работа №2**
  Реализовал API, которое предоставляет CRUD операции для списка товаров (просмотр всех товаров, просмотр товара по id, добавление товара, редактирование товара, удаление товара). Объект товара содержит поля: id, название, стоимость, категория, описание, материал, размеры, цвет, количество на складе.

* **Практическая работа №3**
  1. Протестировал своё API из Практического занятия №2 с помощью Postman (выполнено 3 запроса).
  2. Выбрал открытое API (JSONPlaceholder), изучил документацию и выполнил 5 тестовых запросов. Скриншоты результатов добавлены в папку `screenshots/`.

* **Практическая работа №4**
  Доработал пример: создал два приложения на React и Express, связав их в единую систему — сайт интернет-магазина мебели. Количество товаров — 10. Карточка товара содержит: название, категорию, описание, цену, материал, размеры, цвет, количество на складе.

* **Практическая работа №5**
  Подключил `swagger-jsdoc` и `swagger-ui-express`, описал с помощью JSDoc-аннотаций схему товара (`Furniture`) и все CRUD-операции (`GET`, `POST`, `GET/:id`, `PATCH/:id`, `DELETE`). Убедился, что документация доступна по адресу `/api-docs` и работает в интерактивном режиме.

### Бэкенд

* **Node.js** — среда выполнения JavaScript
* **Express.js** — веб-фреймворк для создания REST API
* **cors** — middleware для обработки CORS-запросов
* **swagger-jsdoc** — генерация OpenAPI спецификации из JSDoc-комментариев
* **swagger-ui-express** — отображение интерактивной документации Swagger

### Фронтенд

* **React.js** — библиотека для построения пользовательского интерфейса
* **Axios** — HTTP-клиент для запросов к API
* **Sass (SCSS)** — препроцессор CSS для стилизации компонентов
* **React Scripts** — сборка и запуск проекта
