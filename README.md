## 🎯 О проекте
Проект создан в рамках практических работ по дисциплине "Фронтенд и бэкенд разработка". 
Представляет собой интернет-магазин видеоигр с возможностью просмотра, добавления и удаления товаров.

## 🛠 Технологии

### Frontend:
- React 18
- Axios для HTTP-запросов
- CSS Modules для стилизации
- SASS (опционально)

### Backend:
- Node.js
- Express
- CORS
- Swagger для документации API

### Инструменты:
- Postman (тестирование API)
- Git (система контроля версий)

## ✨ Функциональность

### Frontend (React):
- Просмотр всех игр в виде карточек
- Добавление новой игры через форму
- Удаление игры
- Адаптивный дизайн
- Индикаторы наличия (мало товара, нет в наличии)

### Backend (Express):
- REST API с 5 эндпоинтами:
  - `GET /games` - получение всех игр
  - `GET /games/:id` - получение игры по ID
  - `POST /games` - добавление новой игры
  - `PATCH /games/:id` - обновление игры
  - `DELETE /games/:id` - удаление игры
- Полная документация API через Swagger UI
